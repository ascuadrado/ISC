Set up can tools in Raspberry Pi:

1. Install can-utils:
sudo apt-get install can-utils

2. Install python-can
pip3 install python-can

3. Install sqlite3
sudo apt-get install sqlite3

4. Install java (for processing - used for display)
sudo apt install openjdk-8-jdk

5. Avoid screen sleep
Append to file: /etc/xdg/lxsession/LXDE-pi/autostart

@xset s off
@xset -dpms

6. Setup of HDMI Screen (only when we want the display to work)
Append to file: /boot/config.txt

# HDMI Screen
hdmi_group=2
hdmi_mode=87
hdmi_cvt=800 480 60 6 0 0 0
hdmi_drive=1

7. Set CAN hardware:
Append to file: /boot/config.txt

dtparam=spi=on
dtoverlay=mcp2515-can0,oscillator=8000000,interrupt=25
dtoverlay=mcp2515-can1,oscillator=8000000,interrupt=24

8. Start CAN Communication (every time the raspberry boots):
(this step is executed in main.py)

sudo ip link set can0 up type can bitrate 250000
sudo ip link set can1 up type can bitrate 500000
